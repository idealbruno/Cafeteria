<h1 align="center">☕ Café - Landing Page com Carrinho Dinâmico</h1>

<p align="center">
  Projeto de cafeteria desenvolvido com HTML, CSS e JavaScript, contendo menu interativo, sistema de carrinho dinâmico e busca de pedidos.
</p>

### 🚀  Funcionalidades

- Página inicial moderna e responsiva
- Menu de produtos
- Carrinho de compras dinâmico
- Sistema de busca
- Layout responsivo para mobile
- Avaliações de clientes
- Integração com Google Maps
- Navegação suave entre seções


###  Tecnologias Utilizadas

  - HTML
  - CSS
  - JAVASCRIPT
  - GOOGLE FONTS
  - ICONS8
  - IMAGENS
 
###  🖥️ Como Executar o Projeto

  1. Clone o repositório:
  2. https://idealbruno.github.io/Cafeteria/
  3. Abra pasta do projeto
  4. Execute o arquivo.
     index.html


###  Funcionalidade do Carrinho
  
  O sistema de carrinho permite:
  - Adicionar produtos
  - Armazenar itens dinamicamente
  - Atualizar valores automaticamente
  - Navegar para a página do carrinho


###   🔍 Sistema de Busca

O campo de busca permite localizar pedidos/produtos rapidamente utilizando JavaScript.
